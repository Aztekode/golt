# Examples

Ejecuta cualquier ejemplo con:

```bash
golt run app.ts
```

Carpetas:

- `hello-http`: servidor HTTP mínimo (puerto 3000)
- `db-users`: ejemplo HTTP async con SQLite + `await db.query()` (puerto 3000)
- `sqlite`: SQLite básico (crea `app.db` en la carpeta del ejemplo)
- `static-spa`: sirve `./public` con fallback SPA (puerto 3000)
