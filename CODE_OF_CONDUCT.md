# Code of Conduct

## Our Pledge

We want Golt to be an open, respectful, and constructive project for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful in disagreements
- Giving actionable technical feedback
- Assuming good intent
- Helping other contributors learn

Examples of unacceptable behavior:

- Harassment or discrimination
- Personal attacks
- Trolling or deliberately disruptive behavior
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues,
and other contributions that violate this Code of Conduct.

## Reporting

Please report unacceptable behavior through the security or maintainer contact
channels documented in `SECURITY.md`.
